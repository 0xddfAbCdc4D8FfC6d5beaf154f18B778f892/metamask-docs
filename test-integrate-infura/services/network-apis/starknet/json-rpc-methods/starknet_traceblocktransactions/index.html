starknet_traceBlockTransactions

Returns the execution trace of all transactions in the specified block. This provides the exact sequence of operations that were performed during the execution of each transaction, and can be useful for debugging.

Parameters​

block_id: [Required] The block parameter object containing one of the following:

  • block_hash: (string) Block hash.
  • block_number: (integer) Decimal block number.
  • One of the string tags latest or pending.

Returns​

A list of execution traces of all transactions in the requested block.

Example​

Replace YOUR-API-KEY with an API key from your Infura dashboard.

Request​

  • cURL
curl https://starknet-mainnet.infura.io/v3/YOUR-API-KEY \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "starknet_traceBlockTransactions",
    "params": {
      "block_id": {
        "block_number": 456600
      }
    }
  }'
